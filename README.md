## 主要功能

1. 团队协作
2. 自定义接口规则
3. 支持常见请求方式
4. 本地内网发送接口(类似postman)
5. 优雅的离线版本导出
6. 细粒度的权限管理
7. 尽可能的提高录入接口效率
8. 开源免费，内网部署安全有保障
